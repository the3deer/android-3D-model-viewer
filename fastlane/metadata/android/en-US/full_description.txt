Open-source 3D model viewer and OpenGL ES 2.0/3.0 demo.

Load and inspect 3D assets while learning how to implement 3D rendering on Android. This app serves as both a utility and an educational resource for developers.

<b>Key Features:</b>
* Support for <b>OBJ, STL, DAE (Collada), GLTF & FBX</b>.
* Modern UI based on <b>Fragments and Navigation Component</b>.
* <b>OpenGL ES 2.0 & 3.0</b> support with advanced shaders.
* <b>Skeletal Animations</b> (DAE & GLTF).
* <b>VR Support</b>: Anaglyph and Stereoscopic modes.
* Interactive tools: Wireframe, Bounding Boxes, Skeleton view, and Ray-casting selection.
* Completely Open Source.

The purpose of this application is to share knowledge about OpenGL and Android development. Check the source code on GitHub: https://github.com/the3deer/android-3D-model-viewer
